## TIPS FOR BEST RESULTS IN NANO BANANA 2
- Set style to **Photorealistic** or **Hyper-realistic**
- Disable any "art" or "illustration" enhancements
- Use **high detail** setting if available
- For interior shots, enable **natural lighting** mode
- Generate at **1792×1024** (hero) or **1024×1024** (square) or **832×1024** (portrait)
- If the first result looks too "AI", add: *"shot on real camera, film grain, real interior, no CGI"*
